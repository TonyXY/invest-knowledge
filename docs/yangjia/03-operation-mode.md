# 03 · 操作模式解析

> 养家心法的核心贡献之一，是将市场划分为不同阶段，并为每个阶段制定了相应的操作模式。

---

## 核心理念：没有绝对的方法

<blockquote class="original-text">
追涨也好，低吸也好，问题不在模式，而在何种阶段应用，以及火候。
</blockquote>

这是养家操作模式的第一原则：**没有绝对正确的方法，只有适合当下阶段的策略**。

<div class="insight-box">
  <div class="insight-title">📌 投资经理解读</div>
  <p>大多数交易者的问题不是不知道方法，而是只掌握了一种方法，然后在所有市场环境中都使用它。</p>
  <ul>
    <li>追涨选手在弱势中频繁被套</li>
    <li>低吸选手在强势中错失主升浪</li>
    <li>价值投资者在熊市中越跌越买</li>
  </ul>
  <p>养家的方法是：建立一个"工具箱"，根据市场阶段选择最合适的工具。</p>
</div>

## 三大操作模式

### 模式一：强势追击（弱势末期 → 强势初期）

<blockquote class="original-text">
强势阶段，就学习钙片兄的做最nb个股的主升浪，可以多拿几天。
</blockquote>

**适用阶段**：市场赚钱效应恢复，热点板块开始形成

**操作要点**：
- 追最强势的龙头股
- 在主升浪中可以持有数天
- 重点出击热点板块的核心标的

**实战案例（养家原文）**：

<blockquote class="original-text">
10月9日重点出击热点黄金股辰州矿业，当天山东黄金、中金黄金、辰州矿业都封死涨停，我选个相对盘小的介入，随后两天轻松赚出。
</blockquote>

### 模式二：超跌反弹（弱势中段）

<blockquote class="original-text">
弱势阶段，可以学炒手的超跌反弹方法，做反弹为主，大盘超跌的时候买超跌的股票，大部分第二天冲高就走，不要有太多想法，如果情况出现不利，亏钱也走。
</blockquote>

**适用阶段**：市场持续下跌，恐慌情绪释放，超跌品种出现

**操作要点**：
- 选择超跌严重的品种
- 预期收益低——大部分第二天冲高就走
- 严格止损——情况不利必须离场

### 模式三：震荡试错（平衡格局）

<blockquote class="original-text">
平衡格局，就部分持有热点板块观察变盘方向，学习先手兄的试错，加仓，向上买，争取顺势而为。
</blockquote>

**适用阶段**：市场方向不明，多空力量均衡

**操作要点**：
- 小仓位试错
- 确认方向后逐步加仓
- 方向向上就做多，方向向下就撤退

## 市场阶段识别

养家将市场分为四个阶段：

| 阶段 | 特征 | 首选策略 |
|------|------|----------|
| **强势阶段** | 赚钱效应持续，热点明确 | 强势追涨龙头，中线持有 |
| **弱势初期** | 强势股开始补跌，亏钱效应出现 | 低吸止损，减少操作 |
| **弱势中段** | 恐慌蔓延，超跌品种增多 | 超跌反弹，快进快出 |
| **弱势末期** | 恐慌释放殆尽，新的热点萌芽 | 追涨新强势股，重仓出击 |

<blockquote class="original-text">
弱势初段低吸止损，弱势中段低吸崩溃，弱势末段追涨强势。
</blockquote>

<div class="insight-box">
  <div class="insight-title">📌 投资经理解读</div>
  <p>这四个阶段的划分借鉴了道氏理论和艾略特波浪理论中关于趋势阶段的思想，但养家的版本完全基于<strong>市场情绪</strong>而非价格形态：</p>
  <ul>
    <li>强势 = 赚钱效应 > 亏钱效应</li>
    <li>弱势 = 亏钱效应 > 赚钱效应</li>
    <li>平衡 = 两者交替出现，方向不明</li>
  </ul>
  <p>观察的指标不是K线形态，而是：涨停板数量、昨日涨停股表现、热点持续性、亏钱效应的扩散程度。</p>
</div>

---

## 火候的重要性

养家反复强调"火候"二字——知道什么阶段用什么方法还不够，**还要知道在什么时机切入**。

<blockquote class="original-text">
火候的问题，我也没法做到精确，大方向把握对了，赚钱还是大概率，多少的问题。
</blockquote>

<div class="insight-box">
  <div class="insight-title">📌 投资经理解读</div>
  <p>"火候"是经验积累的产物，无法通过看书学会。但可以通过以下方法加速积累：</p>
  <ul>
    <li><strong>复盘</strong>：每天回顾自己的交易，记录买入卖出时的市场环境和心理状态</li>
    <li><strong>试错</strong>：在市场关键节点用小仓位验证自己的判断</li>
    <li><strong>总结</strong>：找出自己判断准确和错误的规律，不断迭代</li>
  </ul>
  <p>养家自己说这是"一个动态的变化过程"，所以不要期待能精确预测，能把握大方向就已经足够赚钱了。</p>
</div>

## 实战要点

1. **行情好多做，行情不好多休息**——这是最简单也最难做到的原则
2. **做就做最强**——无论什么模式，选择板块中最强的标的
3. **灵活切换**：强势追涨 → 弱势低吸 → 震荡试错，不要拘泥于一种模式
4. **预期管理**：强势阶段可以多持有几天，弱势阶段预期要放低（第二天冲高就走）
5. **止损及时**：情况不利必须走，不要因为"亏钱了"而犹豫
