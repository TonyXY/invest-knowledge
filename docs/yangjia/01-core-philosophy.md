## 六句话的内在逻辑

六句总纲并非孤立存在，而是一个完整的交易哲学体系：

<div class="insight-box">
  <div class="insight-title">📌 体系逻辑图</div>
  <ol>
    <li><strong>认知层面</strong>（第一句）：市场是情绪驱动的，交易的核心是揣摩情绪</li>
    <li><strong>操作层面</strong>（第二句）：做龙头股，但要在不同阶段采取不同策略</li>
    <li><strong>心态层面</strong>（第三句）：顺势而为，利用市场情绪而非对抗它</li>
    <li><strong>风控层面</strong>（第四句）：仓位取决于机会，不预设顶底</li>
    <li><strong>执行层面</strong>（第五句）：买卖取决于逻辑，而非盈亏比例</li>
    <li><strong>目标层面</strong>（第六句）：追逐人气，因为人气就是资金的方向</li>
  </ol>

## 实战要点

1. **建立情绪观察指标**：每天关注涨停/跌停家数、昨日涨停股今日表现、热点板块持续性
2. **避免先入为主**：不要预设顶部和底部，让市场走势告诉你答案
3. **脱离成本思维**：每次决策都问自己"如果我现在空仓，我会买入这个标的吗？"
4. **理解正反馈**：强势股越涨越有人买，不要因为"涨多了"而卖出——除非上涨逻辑变了
5. **关注人气聚集地**：涨幅榜、成交额排名、讨论热度的交集区域
