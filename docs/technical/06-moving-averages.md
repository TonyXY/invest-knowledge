# 06 · 均线系统

> 如果说技术分析中只允许你用一种指标，那一定是移动平均线。它简单、直观、有效——是趋势跟踪的基石。

---

## 什么是均线

### 通俗解释

**移动平均线（MA）** 就是最近 N 天收盘价的平均值连成的线。

举个例子：
- MA5（5日均线）：把过去5天的收盘价加起来除以5，每天计算一次，连成一条线
- MA10（10日均线）：把过去10天的收盘价加起来除以10，每天计算一次，连成一条线

### 为什么均线有效

均线本质上反映了**市场的平均持仓成本**：

| 均线周期 | 代表什么 | 常见用法 |
|---------|---------|---------|
| MA5 | 最近一周的平均成本 | 短线交易参考 |
| MA10 | 最近两周的平均成本 | 短线趋势 |
| MA20 | 最近一个月的平均成本 | 中期趋势基准 |
| MA60 | 最近一个季度的平均成本 | 中期趋势 |
| MA120 | 最近半年的平均成本 | 长期趋势 |
| MA250 | 最近一年的平均成本 | 牛熊分界线 |

<div class="insight-box">
  <div class="insight-title">📌 投资经理解读</div>
  <p>均线的本质是<strong>"市场的平均成本"</strong>。</p>
  <p>价格在均线上方，说明最近买入的人都在赚钱——赚钱的人倾向于持有，这是上涨的基础。</p>
  <p>价格在均线下方，说明最近买入的人都在亏钱——亏钱的人倾向于卖出，这是下跌的压力。</p>
  <p>这就是均线能成为"动态支撑阻力"的根本原因。</p>
</div>

## 单根均线的用法

### 四个基本判断

| 状态 | 均线位置/方向 | 含义 |
|------|-------------|------|
| 多头 | 价格在均线上方 | 市场偏强，倾向于持有 |
| 空头 | 价格在均线下方 | 市场偏弱，倾向于观望 |
| 趋势向上 | 均线向上倾斜 | 趋势向上，回调可考虑买入 |
| 趋势向下 | 均线向下倾斜 | 趋势向下，反弹可考虑卖出 |

### 手把手：MA20 的实战用法

MA20（20日均线）是最常用的中期趋势均线，建议新手从这根均线开始：

```
1. 价格在 MA20 上方运行 → 中期趋势向上，以持有为主
2. 价格跌破 MA20 → 警示信号，中期趋势可能改变
3. 价格在 MA20 下方运行 → 中期趋势向下，以观望为主
4. 价格突破 MA20 → 可能的转势信号，但需要放量确认
```

**关键判断**：一只股票价格一直在 MA20 上方运行，某天跌破 MA20——这是一个警示信号，说明中期趋势可能改变。这时候需要检查：
- 是否是放量跌破？如果是，趋势改变的概率更高
- 是否是缩量跌破？可能是假跌破，需要继续观察

## 多根均线的组合

当你在图上叠加多根不同周期的均线时，它们之间的排列关系能告诉你当前市场的状态。

### 三种排列形态

**多头排列**——强势上涨

```
MA5 > MA10 > MA20 > MA60
全部向上倾斜
```

含义：短期成本 > 中期成本 > 长期成本，所有买入的人都在赚钱——最强的上涨信号之一。

**空头排列**——弱势下跌

```
MA5 < MA10 < MA20 < MA60
全部向下倾斜
```

含义：短期成本 < 中期成本 < 长期成本，所有买入的人都在亏钱——最弱的下跌信号之一。

**交叉缠绕**——震荡市场

```
均线反复交叉，没有明显方向
```

含义：市场没有趋势，不适合趋势跟踪策略。

### 排列形态对比

| 形态 | 市场状态 | 操作策略 |
|------|---------|---------|
| 多头排列 | 强势上涨 | 持有为主，回调加仓 |
| 空头排列 | 弱势下跌 | 空仓为主，反弹减仓 |
| 交叉缠绕 | 震荡整理 | 高抛低吸，或观望 |

<div class="insight-box">
  <div class="insight-title">📌 投资经理解读</div>
  <p>多头排列是最强的上涨信号之一。在 A 股，当 MA5 > MA10 > MA20 > MA60 形成标准多头排列时，趋势往往能持续一段时间。</p>
  <p>但要注意：多头排列的形成往往已经经历了一段上涨，你不可能在最低点买入。不要等完美入场——多头排列确认后，回调到均线附近时就是相对合理的入场时机。</p>
  <p>同样，空头排列形成时往往已经跌了一段。不要在空头排列中"抄底"——等均线从空头走平再转为多头再考虑。</p>
</div>

## 金叉和死叉

### 定义

- **金叉**：短期均线上穿长期均线——买入信号
- **死叉**：短期均线下穿长期均线——卖出信号

### 常用组合

| 组合 | 含义 | 使用场景 |
|------|------|---------|
| MA5 上穿 MA10 | 短线金叉 | 短线交易 |
| MA5 下穿 MA10 | 短线死叉 | 短线交易 |
| MA10 上穿 MA20 | 中期金叉 | 中期趋势交易 |
| MA10 下穿 MA20 | 中期死叉 | 中期趋势交易 |

### 金叉死叉的局限性

最大的问题：**在震荡市场中会频繁出现假信号**。

```
震荡市场中的走势：
金叉 → 涨了2天 → 死叉 → 跌了3天 → 金叉 → 涨了1天 → 死叉...
```

如果按照信号频繁买卖，你会发现每次都被"左右打脸"。

<div class="insight-box">
  <div class="insight-title">📌 投资经理解读</div>
  <p>金叉的有效性取决于两个因素：</p>
  <ol>
    <li><strong>均线的角度</strong>——金叉发生时，如果均线本身是向上倾斜的，这个金叉的可靠性更高。如果均线是走平的，金叉可能只是一个震荡信号。</li>
    <li><strong>所处的市场阶段</strong>——大趋势整体向上时的金叉更可靠；大趋势向下时的金叉可能只是反弹，追进去容易被套。</li>
  </ol>
  <p>一个简单的过滤方法：只看 MA20 方向向上的时候出现的金叉，忽略 MA20 方向向下的金叉。</p>
</div>

## 均线的实战用法

### 策略一：单根均线趋势跟踪

最简单的趋势跟踪系统：**以 MA20 为多空分界线**。

```
价格在 MA20 上方 → 只做多不做空
价格在 MA20 下方 → 只做空不做多（或空仓）
```

这个策略虽然简单，但能帮你避过大部分下跌行情。

### 策略二：均线组合判断趋势强度

| 条件 | 判断 | 操作 |
|------|------|------|
| 多头排列 + 放量上涨 | 强势上涨 | 积极做多 |
| 多头排列 + 缩量回调 | 正常调整 | 可加仓 |
| 多头排列开始发散 | 加速上涨 | 持有但注意风险 |
| 多头排列开始收敛 | 趋势可能减弱 | 谨慎持有 |
| 空头排列 + 放量下跌 | 加速下跌 | 不要抄底 |
| 空头排列 + 缩量反弹 | 弱势反弹 | 减仓机会 |

### 策略三：均线作为动态支撑阻力

上涨趋势中，回调到某条均线附近获得支撑是常见的买入机会：

```
上升趋势中：
- 强势回调：到 MA10 附近获得支撑
- 正常回调：到 MA20 附近获得支撑
- 深度回调：到 MA60 附近获得支撑（趋势可能改变）
```

### 手把手：新手如何设置均线系统

```
步骤一：打开交易软件，找到均线设置
步骤二：添加四根均线——MA5、MA10、MA20、MA60
步骤三：颜色设置（建议）：
  - MA5：白色（短线最快）
  - MA10：黄色
  - MA20：紫色或红色（最重要的中期线）
  - MA60：绿色（长期线）
步骤四：每天收盘后花2分钟检查均线排列状态
```

四根均线足够了，不需要更多。**新手常见的错误是用太多指标，而不是用好一个指标。**

## 实战要点

<div class="insight-box">
  <div class="insight-title">📌 投资经理解读</div>
  <p>关于均线系统，说三点最实战的：</p>
  <ol>
    <li><strong>均线永远滞后</strong>——它告诉你的是"过去发生了什么"，而不是"未来会发生什么"。不要指望均线给你最完美的买卖点。它能帮你抓住趋势的大部分，但不可能买在最低、卖在最高。</li>
    <li><strong>不同股票用不同的均线</strong>——有些股票对 MA20 反应敏感，有些则对 MA30 更有效。可以复盘历史走势，找出个股"最听话"的那条均线，把它作为你的参考基准。</li>
    <li><strong>均线 + 成交量 = 更可靠的信号</strong>——只看均线不看成交量，就像开车只看方向盘不看油门。突破均线时放量，说明突破有效；缩量突破均线，需要警惕假突破。</li>
  </ol>
