# 02 · K线组合

> 单根K线告诉你"今天发生了什么"，K线组合告诉你"趋势可能要变了"。两根K线比一根可靠，三根比两根更可靠。

单根K线提供的信息有限，因为它只反映了一个时间段的博弈结果。但当多根K线组合在一起时，它们之间就产生了"对话"——前一根的收盘价对后一根的开盘价施加影响，这种前后关系才真正揭示了多空力量的演变过程。

---

## 两根K线的组合

### 吞没形态

吞没形态是两根K线组成的反转信号，分看涨和看跌两种。

**看涨吞没**：出现在下跌趋势末端。第一根是小阴线，第二根是大阳线，且阳线的实体完全覆盖（吞没）了前一根阴线的实体。

```
下跌趋势 → ▏阴 → ██阳█ (阳线实体完全覆盖阴线实体)
```

含义：第一天空方还在主导，第二天多方以更强力量入场，把前一天的所有跌幅全部收复。这是多头反攻的强烈信号。

**看跌吞没**：出现在上涨趋势末端。第一根是小阳线，第二根是大阴线，且阴线的实体完全覆盖了前一根阳线的实体。

```
上涨趋势 → █阳█ → ██阴█ (阴线实体完全覆盖阳线实体)
```

含义：第一天多方还在主导，第二天空方大举反攻，把前一天的涨幅全部吞没。这是空头反击的强烈信号。

**实战场景**：某股票连续跌了5天，每天都是小阴线。第6天突然出现一根大阳线，收盘价比第5天的开盘价还高——这就是典型的看涨吞没形态，可以考虑入场了。

<div class="insight-box">
  <div class="insight-title">📌 投资经理解读</div>
  <p>吞没形态的关键不是实体覆盖，而是背后的情绪反转——前一天的主导方被后一天的主导方完全否定。</p>
  <p>如果你看到看涨吞没，说明多方已经完全收复失地并且表现出更强的进攻欲望。但要注意：吞没形态出现在震荡市中成功率会大打折扣，因为震荡市中"吞没"可能只是随机波动。</p>
</div>

### 孕育线

孕育线和吞没形态相反——第二根K线的实体被第一根K线的实体"包裹"在内。

**看涨孕育**：出现在下跌趋势末端。第一根是大阴线，第二根是小阳线（实体在第一根实体内）。

```
下跌趋势 → ██阴██ → ▏小阳▏ (小阳线实体在大阴线实体之内)
```

含义：大跌之后，第二天价格波动收窄，收出一根小阳线。说明卖出力量在减弱，空方无法再创出新低，多方开始试探性入场。

**看跌孕育**：出现在上涨趋势末端。第一根是大阳线，第二根是小阴线（实体在第一根实体内）。

```
上涨趋势 → ██阳██ → ▏小阴▏ (小阴线实体在大阳线实体之内)
```

含义：大涨之后，第二天价格波动收窄，收出一根小阴线。说明买入力量在减弱，多方无法再推动创新高，空方开始试探性反击。

### 穿刺线 / 乌云盖顶

这两个形态比吞没更温和，但同样是重要的反转信号。

**穿刺线（看涨）**：出现在下跌趋势中。第一天是大阴线，第二天低开但大幅上涨，收盘价进入前一天阴线实体的一半以上位置。

含义：虽然第二天开盘继续下跌（低开），但多方强力反击，收盘进入了前一天的"腹地"，说明多头反击力度很强。越接近前一天的收盘价，反转信号越强。

**乌云盖顶（看跌）**：出现在上涨趋势中。第一天是大阳线，第二天高开但大幅下跌，收盘价进入前一天阳线实体的一半以下位置。

含义：虽然第二天开盘继续上涨（高开），但空方打压使价格大幅回落，收盘跌回前一天的实体内部。说明上涨势头已经被打断。

## 三根K线的组合

### 早晨之星（底部反转信号）

三根K线组成：大阴线 → 小实体（十字星）→ 大阳线

```
大阴线 → 小实体（十字星）→ 大阳线
 ↓下跌   ↓多空平衡       ↓反转上涨
```

**怎么看**：
- 第一天：延续下跌，收大阴线，空方主导
- 第二天：开盘继续下跌，但跌幅收窄，收盘形成小实体或十字星——说明空方力量衰竭
- 第三天：收出大阳线，收盘价进入第一天阴线实体的一半以上——多方全面反攻

**实战场景**：某股从30元一路跌到20元，第1天跌5%收大阴线，第2天只跌0.5%收十字星，第3天涨6%收大阳线——标准的早晨之星。这个形态就像"黎明前的黑暗"——最难熬的时候已经过去。

<div class="insight-box">
  <div class="insight-title">📌 投资经理解读</div>
  <p>早晨之星是可靠性最高的底部反转形态之一。核心逻辑是"下跌三阶段"：恐慌抛售（第一天）→ 抛压耗尽（第二天）→ 多头反攻（第三天）。</p>
  <p>第二天如果是十字星最好，如果是小阳线也可以接受。第三天的大阳线越强势（接近涨停更好），反转信号越可靠。早晨之星后的回调不破第三天阳线的最低价，就是二次确认的加仓点。</p>
</div>

### 黄昏之星（顶部反转信号）

三根K线组成：大阳线 → 小实体（十字星）→ 大阴线

```
大阳线 → 小实体（十字星）→ 大阴线
 ↑上涨   ↓多空平衡       ↓反转下跌
```

**怎么看**：
- 第一天：延续上涨，收大阳线，多方主导
- 第二天：开盘继续上涨，但涨幅收窄，收盘形成小实体或十字星——说明买入力量不足
- 第三天：收出大阴线，收盘价跌入第一天阳线实体的一半以下——空方全面反攻

**实战场景**：某股连续大涨后，第1天涨7%收大阳线，第2天震荡收十字星，第3天跌5%收大阴线——黄昏之星出现，提示顶部可能已经形成。

### 红三兵（上涨持续信号）

连续三根小阳线，每根的收盘价都比前一根高，形成步步高升的阶梯状。

```
█ →  ██ →  ███
```

**怎么看**：
- 每根阳线的实体长度大致相近
- 每一根的收盘价和开盘价都比前一根高
- 上下影线较短，说明上涨过程比较平稳

含义：多方稳步推进，每次回调都被更低位置的买盘接住。这是健康的上涨形态，说明上升趋势正在持续。

**注意区别**：如果三根阳线一根比一根长（最后一天是大阳线），这不是红三兵，而是"加速上涨"——反而要小心短期内涨幅过大、回调风险增加。

### 三只乌鸦（下跌持续信号）

连续三根小阴线，每根的收盘价都比前一根低，形成逐步走低的阶梯状。

```
███ →  ██ →  █
```

**怎么看**：
- 每根阴线的实体长度大致相近
- 每一根的收盘价和开盘价都比前一根低
- 没有明显的下影线（说明没有遇到有效支撑）

含义：空方稳步推进，每次反弹都被更低位置的卖盘打压。这是下跌趋势持续的信号，不要轻易抄底。

## 实操原则

1. **K线组合不是100%准确，需要结合位置判断**：同样的看涨吞没形态，出现在下跌末端（低位）是反转信号，出现在上涨末端（高位）可能只是洗盘。位置是判断K线组合有效性的前提。

2. **组合的位置比组合本身更重要**：同样是十字星 + 大阴线，在顶部叫"黄昏之星"（要卖），在底部只是正常的回调。不要机械地套用形态，先问自己："现在价格处于什么位置？"

3. **实战中优先看大周期**：日线级别的K线组合比60分钟级别可靠，60分钟比15分钟可靠。大周期形成的组合代表更强的市场共识，做中长线交易至少看日线级别。大周期组合的可靠性排序：周线 > 日线 > 60分钟 > 15分钟。

4. **组合需要"确认"**：一个反转形态出现后，不要立即行动——等第二天的K线来确认。比如早晨之星出现后，如果第四天继续上涨，则反转确认；如果第四天又跌回去，则可能是假的。

## 实战要点

1. **组合要"认位置不认名字"**：同样一个"大阳线+十字星"的组合，在低位是反转前的停顿，在高位就是黄昏之星的雏形。永远先判断位置，再看形态名称。

2. **等待确认信号**：反转组合出现后，不要立刻满仓进场。等待第二天的走势确认——如果确认信号出现（比如早晨之星后第四天继续阳线），再加仓不迟。错过第一段涨幅总比抄底在半山腰好。

3. **大周期优先**：实战中优先看日线级别的K线组合，周线更好。15分钟、5分钟级别的K线组合噪音太大，容易产生误判。做中线交易的人，用日线组合判断方向就够了。
