# 07 · 常见技术形态

> 技术形态是市场多空博弈留下的"脚印"。通过识别这些经典形态，你可以提前预判趋势的反转或延续。

---

## 反转形态

反转形态出现在趋势的末端，预示着原有趋势即将结束、新的趋势即将开始。

---

### 1. 头肩顶 / 头肩底

头肩形态是技术分析中最经典、可靠性最高的反转形态。它由一个"头部"、两个"肩膀"和一条"颈线"组成。

#### 头肩顶（顶部反转）

出现在上涨趋势末端，形态如下：

```
        头部（最高）
       ↗        ↘
   左肩          右肩
  ↗                ↘
  ---------------------- 颈线（支撑线）
                       ↘
                        下跌
```

- **左肩**：价格先涨一段，然后回落，形成一个高点
- **头部**：价格再次上涨，创出比左肩更高的新高，然后回落，形成最高点
- **右肩**：价格第三次上涨，但无力突破头部的高点，在左肩附近遇阻回落
- **颈线**：连接左肩和头部之间的回撤低点 + 头部和右肩之间的回撤低点
- **确认**：价格跌破颈线，头肩顶形态正式成立

**如何画颈线：**
1. 找到左肩上涨前的回调低点，记为 A 点
2. 找到头部形成后的回调低点，记为 B 点
3. 连接 A 和 B 两点，这条线就是颈线
4. 价格跌破颈线 + 3% 以上，形态确认

#### 头肩底（底部反转）

出现在下跌趋势末端，是头肩顶的倒置版本：

- **左肩**：价格先跌一段，然后反弹，形成一个低点
- **底部**：价格再次下跌，创出比左肩更低的新低，然后反弹，形成最低点
- **右肩**：价格第三次下跌，但无力跌破底部的最低点，在左肩附近获得支撑反弹
- **颈线**：连接左肩和底部之间的反弹高点 + 底部和右肩之间的反弹高点
- **确认**：价格突破颈线，头肩底形态正式成立

#### 交易策略

| 形态 | 入场点 | 止损位 | 目标位 |
|------|--------|--------|--------|
| 头肩顶 | 跌破颈线时做空 | 头部上方 | 头部到颈线的垂直距离，向下翻一倍 |
| 头肩底 | 突破颈线时做多 | 底部下方 | 底部到颈线的垂直距离，向上翻一倍 |

<div class="insight-box">
  <div class="insight-title">📌 投资经理解读</div>
  <p>头肩形态为什么可靠？它的逻辑是——经过三次上攻（头肩顶）都无法突破，说明原来的上涨趋势已经衰竭，多头力量一次比一次弱。第一次还能创新高（头部），第三次连前高都碰不到（右肩更低），说明多头彻底不行了。颈线就是多头的最后一道防线，跌破颈线意味着多头彻底溃败。</p>
  <p>实战中，头肩底比头肩顶更可靠，因为底部形态通常经历更长时间的酝酿，蓄势更充分。</p>
</div>

### 2. 双重顶 / 双重底（M顶 / W底）

双重顶和双重底是仅次于头肩形态的常见反转形态。

#### M顶（双重顶）

```
        顶1 ≈ 顶2
        ↗  ↘ ↗ ↘
      ↗      ↙
    ↗
  ↗
```

- 价格第一次上攻到某个水平后回落
- 价格第二次再次上攻到同一水平附近，再次回落
- 两个顶部基本在同一价格水平（允许小幅偏差）
- 跌破两个顶之间的回撤低点（颈线），形态确认

#### W底（双重底）

```
  ↘
    ↘
      ↖      ↗
        ↘  ↗
        底1 ≈ 底2
```

- 价格第一次下探到某个水平后反弹
- 价格第二次再次下探到同一水平附近，再次反弹
- 两个底部基本在同一价格水平
- 突破两个底之间的反弹高点（颈线），形态确认

#### 关键特征

| 特征 | 双重顶（M顶） | 双重底（W底） |
|------|--------------|--------------|
| 形成位置 | 上涨趋势末端 | 下跌趋势末端 |
| 两个顶/底高度 | 基本相同 | 基本相同 |
| 成交量 | 第二个顶缩量 | 第二个底缩量 |
| 突破确认 | 跌破颈线 | 突破颈线 |
| 可靠性 | 中等 | 中等 |

**与头肩顶的区别：**
- 双顶的两个顶部基本在同一水平，没有明显的"中间高两边低"
- 头肩顶的中间顶（头部）明显高于左右两肩
- 双顶的形态时间通常比头肩顶短

<div class="insight-box">
  <div class="insight-title">📌 投资经理解读</div>
  <p>双顶/双底形态中，第二个顶必须有缩量特征。如果第二个顶放量突破了前高，那就不是双顶，而是上涨中继。同样的，W底的第二个底必须缩量，说明抛压衰竭。</p>
  <p>实战口诀：放量突破或放量跌破是真实信号，缩量的假突破/假跌破大概率是陷阱。</p>
</div>

### 3. V型反转

V型反转是最难把握的反转形态——它来得快、来得猛，没有任何过渡信号。

#### 特征

```
    ↗  V型顶
  ↗
↗
  ↘
    ↘
      ↘  V型底
```

- **速度快**：通常3-5个交易日完成反转
- **幅度大**：涨跌幅度往往超过10%
- **没有过渡**：没有盘整、没有二次确认，直接反转

#### 应对策略

| 策略 | 说明 |
|------|------|
| 不猜V型底 | V型底没有左侧确认信号，猜底等于赌博 |
| 等右侧确认 | 等价格反弹后出现回踩不破前低，再考虑入场 |
| 分批入场 | 先轻仓试错，确认后再加仓 |
| 严格止损 | V型反转失败的概率也不低，必须有止损 |

<div class="insight-box">
  <div class="insight-title">📌 投资经理解读</div>
  <p>V型反转是散户最容易被甩下车的情况——恐慌割肉在V型底，追高买入在V型顶。应对V型的核心不是"抓住它"，而是"不被它伤害"。等右侧信号确认后再行动，虽然会错过一小段行情，但换来的是更高的安全边际。</p>
  <p>记住：你不需要抓住每一段行情，你只需要抓住你能把握的那一段。</p>
</div>

## 持续形态

持续形态出现在趋势运行的中段，意味着趋势暂时休整，休整结束后将继续沿原方向运行。

---

### 4. 旗形整理

旗形整理是上涨或下跌过程中出现的小幅反向整理形态，形状像一面旗帜。

#### 形态特征

```
  上涨旗形：               下跌旗形：
  
  ↗  ↗↗                   ↘  ↘↘
 ↗ ↗↗ ↘↘                 ↘ ↘↘
↗ ↗↗   ↘↘               ↘ ↘↘
```

- **上涨旗形**：快速上涨后出现小幅下跌整理，整理通道向下倾斜
- **下跌旗形**：快速下跌后出现小幅反弹整理，整理通道向上倾斜
- 形态像平行四边形的旗帜
- 成交量在整理期间萎缩，突破时放大

#### 交易策略

- 在旗形整理的末端、价格突破整理区间时跟随
- 上涨旗形：突破上轨时买入
- 下跌旗形：跌破下轨时卖出
- 止损设在整理区间的另一端

<div class="insight-box">
  <div class="insight-title">📌 投资经理解读</div>
  <p>旗形整理是"趋势的加油站"。一波快速上涨/下跌后，多空双方都需要喘口气。赢利的想兑现，观望的想入场——双方在这个区间内完成换手。换手充分后，趋势继续。</p>
  <p>如果旗形整理的时间过长（超过3周），它就不再是旗形，可能演变为顶部或底部形态。旗形的核心是"短暂整理"。</p>
</div>

### 5. 三角形整理

三角形整理是持续形态中最常见的类型。价格在逐渐收窄的区间内运行，最终选择方向突破。

#### 三种三角形

| 类型 | 形状 | 特征 | 突破方向 |
|------|------|------|---------|
| 上升三角形 | 顶部水平，底部抬高 | 买方越来越强，每次回落都能在更高的位置获得支撑 | 大概率向上 |
| 下降三角形 | 顶部降低，底部水平 | 卖方越来越强，每次反弹都在更低的位置遇阻 | 大概率向下 |
| 对称三角形 | 顶部降低，底部抬高 | 多空双方力量趋于平衡，方向不确定 | 方向不确定 |

```
上升三角形：        下降三角形：        对称三角形：
  顶—顶—顶          顶—顶                 顶—顶
  ↗↘↗↘            ↘  ↘↘               ↘  ↘↘
 ↗  ↘  ↘         ↘  ↘  ↘              ↗  ↗  ↗
↗    ↘  ↘         底—底—底            ↗    ↗    ↗
      ↘                                底—底
```

#### 交易策略

- **上升三角形**：在价格突破上边水平线时买入
- **下降三角形**：在价格跌破下边水平线时卖出
- **对称三角形**：等待方向选择，突破哪个方向就跟哪个方向
- 三角形整理的突破通常发生在形态的2/3到3/4处（即从最宽处算起，走完约70%的行程时）

<div class="insight-box">
  <div class="insight-title">📌 投资经理解读</div>
  <p>三角形整理的本质是多空双方力量趋于平衡，等待一方打破僵局。</p>
  <p>通常三角形末端的突破力度会减弱——因为到末端时，价格波动的幅度已经很小，说明双方力量已经很接近，即便突破，后续动力也有限。所以三角形中段的突破最可靠，力度最大。</p>
  <p>一个经验法则：三角形整理的时间越短、突破越早，后续行情越大。</p>
</div>

## 形态交易的通用原则

1. **形态越大、时间越长，突破后的行情越大**。一个耗时3个月的头肩底比耗时3天的头肩底更有价值，因为蓄势更充分。

2. **成交量验证**。反转形态需要放量确认——头肩底突破颈线时需要放量，M顶跌破颈线时需要放量。没有成交量配合的突破很可能是假突破。

3. **位置比形态本身更重要**。同样的头肩顶出现在高位（经过大幅上涨后）和出现在低位（刚启动不久），含义完全不同。高位头肩顶是真正的顶部信号，低位头肩顶可能是洗盘后的中继形态。

4. **不要提前入场，等形态完成后再操作**。形态在没有确认之前都只是"可能"。形态交易最忌讳"猜"——"我觉得这个头肩底快形成了，先买进去"。要等形态确认（突破颈线）后再行动。

| 原则 | 一句话总结 |
|------|-----------|
| 形态规模 | 越大越可靠 |
| 成交量 | 突破必须有量配合 |
| 位置 | 高位低位含义不同 |
| 确认 | 等形态完成，不猜 |

## 实战要点

1. **先判断趋势方向，再识别形态**。在一个上升趋势中，你看到的持续形态比反转形态更可信。逆趋势的形态信号可靠性打五折。

2. **形态识别不要太追求完美**。实际走势中很少有教科书式的完美形态——有些许偏差是正常的。关键的判断标准是"是否符合形态的核心逻辑"，而不是"和教科书上的图一不一样"。

3. **成交量是你的好帮手**。成交量可以帮你区分"真正的形态突破"和"假突破"。真突破放量，假突破缩量。

4. **多周期确认**。如果日线级别出现了头肩底形态，去看周线是否也支持上涨。多周期共振的形态信号更加可靠。

5. **形态不是万能的**。技术形态的识别有一定主观性，不同的分析师可能画出不同的颈线。结合其他技术分析工具（趋势线、均线、支撑阻力）一起使用，效果更好。
