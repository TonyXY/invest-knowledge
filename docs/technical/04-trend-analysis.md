# 04 · 趋势判断

> "趋势是你的朋友"——这可能是技术分析里最重要的一句话。但前提是，你要先学会识别趋势。

---

## 趋势的三种方向

趋势只有三种方向：**上升**、**下降**、**横盘**。没有第四种。

### 上升趋势

上升趋势的定义：**更高的高点和更高的低点**（higher highs + higher lows）。

```
价格
↑       高点3
↑      ↗
↑    ↗  低点3
↑  ↗
↑ ↗  低点2        高点3 > 高点2 > 高点1
↑↗              低点3 > 低点2 > 低点1
  低点1
└─────────────────────────→ 时间
```

每一次上涨的高点都比前一次高，每一次回调的低点也比前一次高——这就是上升趋势。

股票在上升趋势中运行时，买入的胜率天然更高——因为趋势本身在帮你。

### 下降趋势

下降趋势的定义：**更低的高点和更低的低点**（lower highs + lower lows）。

```
价格
  高点1
↘
  ↘   高点2
    ↘
      ↘   高点3
        ↘
          ↘   低点3
  低点1      ↗
          低点2          高点1 > 高点2 > 高点3
                         低点1 > 低点2 > 低点3
  └─────────────────────────→ 时间
```

每一次反弹的高点都比前一次低，每一次下跌的低点也比前一次低——这就是下降趋势。

在下降趋势中买入，即使你买的价格已经"很低"了，后面可能还有"更低"。

### 横盘震荡

横盘震荡的定义：价格在一个水平区间内来回运动，没有明确的方向。

```
价格
  高点1   高点2   高点3
━━━━━━━━━━━━━━━━━━━ 压力位
  ▏  ▏  ▏  ▏  ▏
  ▏  ▏  ▏  ▏  ▏
━━━━━━━━━━━━━━━━━━━ 支撑位
  低点1   低点2   低点3
└─────────────────────────→ 时间
```

高点和低点都在差不多的位置，没有明显的抬升或降低——这是蓄力阶段，突破方向决定后续走势。

## 如何画趋势线

趋势线是最简单但最有效的趋势识别工具。

### 手把手画上升趋势线

1. 找到K线图上依次抬高的**低点**（至少两个）
2. 用一条直线连接这两个低点
3. 如果第三个低点也落在这条线上（或附近），这条趋势线就非常有效

```
上升趋势线画法：
             高点3
            ↗
     高点2 ↗  低点3（第三个点刚好落在线上）
    ↗    ↗
  ↗ ↗  低点2
↗ ↗
低点1           ←-- 连接低点1和低点2，延伸出去
```

### 手把手画下降趋势线

1. 找到K线图上依次降低的**高点**（至少两个）
2. 用一条直线连接这两个高点
3. 如果第三个高点也落在这条线上（或附近），这条趋势线就非常有效

```
下降趋势线画法：
高点1
  ↘  ↘
    ↘   高点2
      ↘
        ↘   高点3（第三个点刚好落在线上）
          ↘
            ↘
```

### 趋势线画法的三个要点

- **至少两个点确认**——两个点画一条线，这是最基础的要求
- **三个点更可靠**——如果价格第三次碰到趋势线后反弹，说明这条线被反复验证，有效性高
- **触及次数越多越重要**——一条被触碰过4-5次的趋势线，比只被触碰2次的趋势线更有参考价值

<div class="insight-box">
  <div class="insight-title">投资经理解读</div>
  <p>趋势线画法没有绝对标准——同一条K线图，不同的人可能画出不同的趋势线。关键在于"<strong>多数人都在这么画</strong>"。</p>
  <p>为什么？因为大家都在看同样的位置，这些位置就成了心理关口。当一个明显的低点连接成的趋势线被突破时，所有看着这条线的人都会做出相同的反应——这就是趋势线有效的原因。</p>
  <p>所以画趋势线的原则是：<strong>画大家都会看到的那条线</strong>，而不是你"觉得最合理"的那条线。</p>
</div>

## 趋势的三种状态和时间框架

趋势是分时间级别的。日线图的上升趋势，在周线图上可能只是下降趋势中的一个反弹。

### 三种时间框架

| 趋势类型 | 时间跨度 | 适合的投资者 |
|:--------|:---------|:------------|
| 主要趋势（长期） | 数月至数年 | 长线投资者、价值投资者 |
| 次级趋势（中期） | 数周至数月 | 波段交易者 |
| 小趋势（短期） | 数天至数周 | 短线交易者、日内交易者 |

### 手把手：区分操作周期

**做短线**的看盘周期：
- 用日线判断方向
- 用60分钟线找买卖点

**做中线**的看盘周期：
- 用周线判断方向
- 用日线找买卖点

**做长线**的看盘周期：
- 用月线判断方向
- 用周线找买卖点

关键原则：**大周期定方向，小周期定入场点**。如果你的周线图是上升趋势，日线图的回调就是买入机会；反之，如果周线图是下降趋势，日线图的反弹就是卖出机会。

## 趋势反转的识别

趋势反转是交易中最重要、也是最难判断的问题。

### 上升趋势结束的信号

上升趋势结束需要同时满足两个条件：

1. **跌破上升趋势线**——价格跌破了之前连接低点的上升趋势线
2. **无法创新高**——反弹时的高点没有超过前一个高点

```
上升趋势反转示意：
          前高点
        ↗     ↘
      ↗         ↘（反弹不能创新高）
    ↗            ×  ←-- 跌破趋势线
  ↗          ↘
↗        ↘
原始趋势线───────突破
```

当这两个条件同时出现，说明上升趋势可能已经结束，趋势可能转为下降或横盘。

### 下降趋势结束的信号

下降趋势结束也需要同时满足两个条件：

1. **突破下降趋势线**——价格向上突破了之前连接高点的下降趋势线
2. **无法创新低**——回调时的低点没有低于前一个低点

```
下降趋势反转示意：
             突破趋势线
          ↗
        ↗   ×  ←-- 回踩不创新低
      ↗
    ↗        前低点
原始趋势线───────
↗
```

### 最重要的原则

<div class="insight-box">
  <div class="insight-title">投资经理解读</div>
  <p><strong>不要猜顶底，等趋势自己告诉你。</strong></p>
  <p>这是初学者最容易犯的错误——看到涨了很多就觉得"到顶了"，看到跌了很多就觉得"到底了"。</p>
  <p>事实上，在趋势没有明确的反转信号之前，它更大概率会沿着原来的方向继续。你"觉得"到顶和实际到顶之间，可能差了50%的涨幅。</p>
  <p>正确的做法是：不预测顶底，而是等趋势反转信号出现后再行动。虽然会错过"最高点"和"最低点"，但你买在的是确认上涨趋势的位置，卖在的是确认下跌趋势的位置——长期来看，这种做法的胜率远高于"抄底逃顶"。</p>
</div>

## 不同趋势下的策略

### 上升趋势：回调买入，持有为主

上升趋势中的策略核心是**做多**：

1. 已经持有的：继续持有，不要因为"涨太多了"就卖
2. 想要买入的：等回调到趋势线或均线附近时买入，不要在连续大涨后追高
3. 止盈：用移动止盈法，让利润奔跑（参见03篇的移动止盈方法）

### 下降趋势：反弹卖出，空仓观望

下降趋势中的策略核心是**防御**：

1. 已经持有的：每一次反弹都是减仓的机会
2. 想要买入的：耐心等待趋势反转信号出现
3. 不要"抄底"——在下降趋势中买在"最低点"的概率极低，抄底抄在半山腰才是常态

### 横盘震荡：高抛低吸，等待突破

横盘震荡中的策略核心是**波段操作**：

1. 靠近支撑位买入
2. 靠近压力位卖出
3. 突破后顺势加仓或止损
4. 不放量突破之前，不判断方向

| 趋势方向 | 操作策略 | 仓位建议 | 核心原则 |
|:--------|:---------|:--------|:---------|
| 上升趋势 | 回调买入，持有为主 | 重仓 | 顺势而为 |
| 下降趋势 | 反弹卖出，空仓观望 | 轻仓或空仓 | 现金为王 |
| 横盘震荡 | 高抛低吸，等待突破 | 中等仓位 | 不预测方向 |

<div class="insight-box">
  <div class="insight-title">投资经理解读</div>
  <p>"趋势是你的朋友"——这不是一句口号，而是一个经过统计验证的事实。</p>
  <p>在上升趋势中做多和在下降趋势中做空，成功概率天然就有差别。不是因为你的分析能力提高了，而是因为趋势本身在帮你。</p>
  <p>很多人亏钱的根源不是选股选错了，而是选对了股票但在错误的趋势里操作——上升趋势中的股票，因为"怕回撤"早早卖出；下降趋势中的股票，因为"觉得便宜了"不断抄底加仓。</p>
  <p><strong>永远不要逆着趋势操作。</strong>这句话值得你写下来贴在屏幕上。</p>
</div>

## 实战要点

- **先看大周期，再看小周期**——打开K线图的第一件事不是看今天涨了没有，而是先看周线和月线，确定大方向再看日线。大周期定方向，小周期定入场点
- **趋势线不是预言，是纪律**——画趋势线不是为了"预测"它不会被突破，而是为了"如果被突破我该怎么办"。趋势线被跌破就该减仓，被突破就该跟进，这就是交易纪律
- **不要和趋势辩论**——你不需要理解"为什么涨"，你只需要知道"现在在涨"。趋势一旦形成，会持续比大多数人预期更长的时间
- **趋势反转需要时间确认**——一两天的大涨不一定是反转，可能是反弹。等待两个条件同时成立（突破趋势线 + 无法创新低/新高），不要急于下结论
- **不要在横盘震荡中判断方向**——横盘就是横盘，它既不是"蓄力上涨"也不是"酝酿下跌"。等突破方向出来后再说。在横盘区间内做方向预判，胜负各半，没有意义
