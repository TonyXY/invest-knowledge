# 05 · 支撑与阻力

> 支撑和阻力是技术分析中最基础、最实用的概念之一。理解为什么价格会在某些位置停下来，你就能看懂市场的"压力"和"支撑"，从而制定清晰的买卖计划。

---

## 支撑和阻力是什么

### 通俗定义

- **支撑**：价格跌到某个区域就"跌不动了"，然后反弹——这个区域就是支撑
- **阻力**：价格涨到某个区域就"涨不动了"，然后回落——这个区域就是阻力

可以把支撑想象成"地板"，阻力想象成"天花板"。

### 为什么会形成支撑和阻力？

核心原因是**心理因素**——市场参与者对某些价格位置有集体记忆：

| 价格位置 | 为什么成为支撑/阻力 |
|---------|------------------|
| 前期低点 | 之前在低位没买的人后悔了，想等价格回到这里买入 |
| 前期高点 | 之前在涨到这里被套的人终于解套了，想卖出 |
| 整数关口 | 10元、20元、100元——人的大脑偏爱整数，在这些位置买卖双方都会变得敏感 |
| 密集成交区 | 大量交易发生的区域，有最多的"记忆"——套牢盘和获利盘聚集 |

<div class="insight-box">
  <div class="insight-title">📌 投资经理解读</div>
  <p>支撑和阻力本质上不是"线"，而是<strong>"区域"</strong>。</p>
  <p>精确到小数点后两位的支撑位是没有意义的——市场不会因为你的画线就精确到那个位置反弹。关键是价格到达这个区域时的市场反应：成交量是否放大、K线形态是否有止跌或遇阻信号。</p>
  <p>把支撑阻力理解为一个"模糊的区域"（比如10.00-10.20），而不是一条"精确的线"（比如10.05），你的交易会从容很多。</p>
</div>

## 如何识别关键支撑阻力位

### 手把手方法

**第一步：找出前期高点和低点**

打开K线图，从最近的走势开始，找出明显的波段高点和波段低点。把这些位置标出来——它们是最基本的支撑和阻力。

**第二步：标记整数关口**

A股中整数关口尤其明显：10元、20元、50元、100元……这些位置即使没有技术含义，心理含义也非常强。

**第三步：寻找密集成交区**

切换到成交量图，找成交量明显放大的价格区间。这些位置有大量的"筹码被交换"——意味着有大量的市场参与者在这个位置有买卖记忆。

**第四步：画趋势线**

- 上升趋势中，连接两个及以上低点得到上升趋势线——它是**动态支撑**
- 下降趋势中，连接两个及以上高点得到下降趋势线——它是**动态阻力**

### 多周期确认

同一个支撑阻力位，在越多时间周期上被确认，它的有效性就越强：

| 周期级别 | 被确认 | 有效性 |
|---------|-------|-------|
| 只在15分钟图 | 出现一次反弹 | 弱 |
| 在日线+周线 | 同时出现 | 较强 |
| 在日线+周线+月线 | 同时出现 | 非常强 |

## 支撑阻力的互换原理

这是支撑阻力中最重要、最实用的原理：

> **阻力一旦被有效突破，就变成支撑**
> **支撑一旦被有效跌破，就变成阻力**

简单记忆：**"角色互换"**。

### 举例说明

一只股票长期在15-20元之间震荡：
- 20元是阻力（多次涨到这里就回落）
- 15元是支撑（多次跌到这里就反弹）

**情景一：突破阻力**

股价放量突破20元，站稳了。这时候：
- 20元变成了**支撑**
- 如果后续股价回调到20元附近，这是一个买入机会
- 因为之前在这里卖出的人后悔了，想买回来

**情景二：跌破支撑**

股价跌破15元，没有反弹。这时候：
- 15元变成了**阻力**
- 如果后续股价反弹到15元附近，这是一个卖出机会
- 因为之前在这里买入的人被套了，想借反弹卖出

<div class="insight-box">
  <div class="insight-title">📌 投资经理解读</div>
  <p>支撑阻力互换原理非常实用——当你看到一只股票突破了前高，这个前高就是后续回调的<strong>支撑位</strong>。这就给了你一个清晰的加仓位置。</p>
  <p>反过来，如果一只股票跌破了前期低点，这个点位就是你后续反弹减仓的位置。</p>
  <p>这个原理帮你把"已经发生过的走势"转化为"未来的交易计划"。</p>
</div>

## 突破和假突破

不是所有价格冲破支撑阻力都是有效的。你需要学会区分"有效突破"和"假突破"。

### 有效突破的特征

| 特征 | 说明 |
|-----|------|
| 放量 | 突破时成交量明显放大，说明有资金在参与 |
| 站稳 | 连续2-3天在突破位上方/下方，而不是一天游 |
| 幅度 | 突破幅度超过3%（或其他设定的阈值），而不是刚好触碰 |

### 假突破的特征

| 特征 | 说明 |
|-----|------|
| 缩量突破 | 成交量没有配合，突破缺乏资金支持 |
| 天量折返 | 放出巨大成交量后快速跌回，说明主力在出货 |
| 一日游 | 今天突破明天就回来，没有持续性 |

### 如何应对假突破

最实用的策略：**不要追突破，等回踩确认**。

### 手把手操作流程

```
1. 价格突破阻力位 → 不买，观察
2. 价格回踩阻力位（此时已变为支撑）→ 重点关注
3. 价格在阻力位上方企稳，没有跌回去 → 买入
4. 止损设在新支撑位下方一点
```

这样做的好处是：假突破会被过滤掉，因为假突破往往不会回踩——它会直接跌回去。

## 如何用支撑阻力制定交易计划

### 三种常见策略

**策略一：在支撑位附近买入**

- 当价格回调到关键支撑位附近时，寻找买入信号
- 止损设在支撑位下方一点（比如3-5%）
- 目标位是上方最近的阻力位

**策略二：在阻力位附近卖出/减仓**

- 当价格上涨到关键阻力位附近时，考虑部分止盈
- 如果同时出现遇阻信号（长上影线、成交量萎缩），可以考虑全部卖出

**策略三：突破后加仓**

- 价格有效突破阻力位后，加仓
- 以原阻力位（现支撑位）为止损位
- 目标位是下一层阻力

### 完整交易计划示例

```
股票：某票当前在17元
关键支撑：15元（前期低点+整数关口）
关键阻力：20元（前期高点+整数关口）

买入计划：
- 价格回调到15-16元区间，出现止跌信号 → 买入
- 止损设在14.5元

加仓计划：
- 价格有效突破20元（放量+站稳2天）→ 加仓
- 以19.5元为止损

卖出计划：
- 价格涨到20元附近遇阻 → 减仓一半
- 剩余仓位跟踪止损
```

## 实战要点

<div class="insight-box">
  <div class="insight-title">📌 投资经理解读</div>
  <p>关于支撑阻力，说三点最实战的：</p>
  <ol>
    <li><strong>不要追求精确</strong>——支撑阻力是区域，你画在 10.02 还是 10.05 没有本质区别。关键看价格到这个区域时的表现。</li>
    <li><strong>级别越高越有效</strong>——月线级别的支撑阻力远强于日线级别的。大周期的支撑阻力一旦被突破，往往意味着更大级别的趋势转变。</li>
    <li><strong>支撑和阻力是用来"应对"的，不是用来"预测"的</strong>——你不需要预测价格是否会突破某个阻力位。你只需要知道：如果突破，我怎么做；如果不突破，我又怎么做。</li>
  </ol>
