# 01 · K线基础

> K线是技术分析最基础的语言。一根K线包含了一段时间内的全部博弈信息。

## K线的四个价格要素

一根K线由四个价格构成，它们记录了特定时间段内的多空博弈：

| 价格要素 | 英文 | 代表含义 |
|---------|------|---------|
| 开盘价 | Open | 该时间段开始时的第一笔成交价 |
| 收盘价 | Close | 该时间段结束时的最后一笔成交价 |
| 最高价 | High | 该时间段内达到的最高成交价 |
| 最低价 | Low | 该时间段内达到的最低成交价 |

把这四个价格画成一根竖线加两个横线，就是K线。

<div class="insight-box">
  <div class="insight-title">📌 投资经理解读</div>
  <p>四个价格中，收盘价最重要——它代表了多空双方经过一天博弈后的最终结果。开盘价反映的是隔夜情绪和集合竞价的共识，但收盘价才是"谁赢了"的答案。</p>
  <p>专业交易员看K线，第一眼永远是看收盘价相对于开盘价的位置，而不是看最高最低。</p>
</div>

## 阴线和阳线

K线分两种颜色（A股软件通常是红绿，海外是绿红，颜色只是习惯不同）：

**阳线**（实体红色/空心）：收盘价 > 开盘价，代表买入力量占优，价格被推高。

**阴线**（实体绿色/实心）：收盘价 < 开盘价，代表卖出力量占优，价格被打压。

### 实体的含义

实体是开盘价和收盘价之间的部分。实体的长短反映了力量的强弱：

- **实体长**：一方力量压倒性占优。阳线实体长 = 买入力量强；阴线实体长 = 卖出力量强
- **实体短**：多空力量接近，没有一方能主导走势

### 影线的含义

影线是价格超出实体范围的部分（上影线延伸到最高价，下影线延伸到最低价）：

- **上影线长**：价格曾经冲高但被打压下来，说明上方抛压重，空方在阻止上涨
- **下影线长**：价格曾经下跌但被拉回，说明下方支撑强，多方在托底
- **影线短或没有**：价格在区间内平稳运行，没有遇到明显的阻力或支撑

判断方法：
```
上影线长度 = 最高价 - max(开盘价, 收盘价)
下影线长度 = min(开盘价, 收盘价) - 最低价
实体长度 = |收盘价 - 开盘价|
```

## 八种常见单根K线形态

| 形态 | 外观特征 | 含义 | 实战提示 |
|------|---------|------|---------|
| **大阳线** | 无影线或极短影线，实体很长 | 买盘极强，多方完胜 | 出现在低位是启动信号，高位需警惕加速赶顶 |
| **大阴线** | 无影线或极短影线，实体很长 | 卖盘极强，空方完胜 | 出现在高位是见顶信号，低位可能是恐慌最后一跌 |
| **小阳线/小阴线** | 实体短，影线短 | 多空平衡，方向不明 | 行情盘整期常见，观望或等待方向选择 |
| **十字星** | 实体极短（近乎一条横线），有上下影线 | 多空势均力敌，博弈激烈 | 高位十字星 = 见顶警示，低位十字星 = 见底可能 |
| **锤子线（长下影线）** | 下影线很长，实体短小，上影线短或无 | 下跌中被拉起，下方支撑强 | 出现在下跌末端是反转信号，叫"锤子线" |
| **倒锤子（长上影线）** | 上影线很长，实体短小，下影线短或无 | 冲高回落，上方压力大 | 出现在上涨末端是警示信号，叫"倒锤子" |
| **T字线** | 无上影线，下影线长，实体小 | 先跌后涨，支撑强劲 | 低位的T字线通常是止跌信号 |
| **一字线** | 无实体无影线，只有一条横线 | 极端行情，开盘即封板 | 涨停一字线代表买不到，跌停一字线代表卖不出 |

### 详细解读

**大阳线**：开盘后价格一路走高，最终以接近最高价收盘，说明当天买入方全程主导。低位出现大阳线，往往是行情启动的信号。但如果已经在高位连续大涨后出现大阳线，要警惕可能是"加速赶顶"——即最后一波拉升后主力出货。

**大阴线**：开盘后价格一路走低，收盘在最低点附近，说明卖出方完全控制局面。高位大阴线是强烈的见顶信号。低位大阴线则比较复杂——可能是恐慌性杀跌（最后一跌），也可能是下跌中继。

**小阳线/小阴线**：实体短小，说明多空双方力量接近。在盘整期这种K线很常见，没有明确方向，建议观望。

**十字星**：开盘价和收盘价几乎相同，说明多空双方经过一天博弈后打成了平手。关键要看十字星出现的位置：高位十字星说明上涨遇到了强力反抗，可能是顶部；低位十字星说明下跌遇到了支撑，可能是底部。

**锤子线（长下影线）**：价格盘中大幅下跌，但收盘时几乎收复全部失地。形状像一把锤子——所以得名。在下跌趋势末端出现锤子线，说明有资金在低位买入托底，可能是止跌信号。

**倒锤子（长上影线）**：价格盘中大幅冲高，但收盘跌回原点。形状像倒过来的锤子——所以得名。在上涨趋势末端出现倒锤子，说明高位抛压重，可能是见顶信号。

**T字线**：开盘后价格下跌，但被强力拉起最终收在开盘价。形状像个"T"字。低位T字线和锤子线含义类似——支撑强，止跌信号。

**一字线**：开盘即封板（涨停或跌停），全天没有成交或极少成交，所以只有一条横线。涨停一字线表示买盘极强、无人卖出；跌停一字线表示卖盘极强、无人买入。一字线出现时，普通投资者基本没有操作空间。

## Live Example

```
假设今天某只股票的四个价格分别是：
开盘10元，收盘10.6元，最高10.8元，最低9.9元

问：这是阳线还是阴线？实体多长？上下影线分别多长？

答：收盘10.6 > 开盘10，所以是阳线。

实体 = 10.6 - 10 = 0.6元
上影线 = 10.8 - 10.6 = 0.2元
下影线 = 10 - 9.9 = 0.1元

结论：这是一根带小上影线的中阳线，说明买盘力量占优，
但10.8附近有抛压，价格没能收在最高点。
```

另一个例子：
```
开盘50元，收盘47.5元，最高51元，最低47元

问：这是阳线还是阴线？实体多长？上下影线呢？

答：收盘47.5 < 开盘50，所以是阴线。

实体 = 50 - 47.5 = 2.5元
上影线 = 51 - 50 = 1元
下影线 = 47.5 - 47 = 0.5元

结论：这是一根带上影线的中阴线。空方主导，多方
曾试图拉升到51元但被击退，最终收在低位。
```

## 实战要点

1. **先看方向再看力度**：拿到一根K线，第一眼判断是阳还是阴（谁赢了），再看实体长短（赢了多少），最后看影线（有没有阻力/支撑）。

2. **关注影线暴露的信息**：影线比实体更有价值——它告诉你价格在哪里受阻、在哪里获得支撑。比如一根大阳线带长上影线，说明虽然收了阳线，但上方已经有强大的卖盘在等着了。

3. **同样形态在不同位置意义不同**：低位出现锤子线可能是反转信号，高位出现同样的锤子线可能只是回调洗盘。K线的含义永远要结合价格所处的相对位置来判断。
