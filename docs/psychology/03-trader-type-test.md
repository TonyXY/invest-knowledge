## 各类型速查

### 🟢 恐惧型

**典型表现：** 止损特别快但卖在最低点，盈利一点就跑，看准了也不敢下单。

**解法：** 降低仓位到舒服的水平，用分批建仓代替一把梭，接受交易是概率游戏。

### 🔴 贪婪型

**典型表现：** 不止损死扛，盈利不愿卖，经常重仓出击。

**解法：** 强制挂止损单，设置仓位上限（单票不超20%），盈利后分批止盈。

### 🟡 冲动型

**典型表现：** 看到涨就追跌就割，频繁交易，没有固定系统。

**解法：** 降低交易频率的硬约束，下单前写交易理由，关注执行而非盈亏。

### ⚫ 偏执型

**典型表现：** 死扛跟市场较劲，研究过度，过度自信。

**解法：** 设置硬性止损并拉黑一周，做多空双向思考，用概率思维替代判断思维。

### 🌀 混合型（大多数人的情况）

在不同市场环境下表现出不同倾向。建议先找出当前最致命的一个问题，集中解决。

  <div class="insight-title">📌 投资经理解读</div>
  <p>大多数人不是纯粹的一种类型，而是在不同市场环境下表现出不同的倾向。</p>
  <p>比如：</p>
  <ul>
    <li>震荡市中容易变成冲动型（频繁交易）</li>
    <li>趋势市中容易变成贪婪型（不止损）</li>
    <li>连续亏损后容易变成恐惧型（不敢下单）</li>
    <li>连续盈利后容易变成偏执型（过度自信）</li>
  </ul>
