# 03 · 止盈止损设置

> 养家说"永不止损，永不止盈"——他的意思不是不设止损，而是反对机械地按点数止损。

---

## 止损：逻辑坏了就走

养家说"永不止损"，他反对的是机械止损（跌5%就割）。他的做法是：**逻辑不成立就卖出，跟亏了多少无关。**

所以止损的关键不是设几个点，而是**事先想清楚你的买入逻辑是什么**：

| 买入逻辑类型 | 止损条件 |
|---|---|
| 追龙头（情绪驱动） | 龙头倒了，板块退潮 → 走 |
| 超跌反弹（技术驱动） | 反弹不及预期，或者继续破位 → 走 |
| 趋势跟随（突破买入） | 跌破突破K线的最低价 → 走 |
| 基本面驱动（价值逻辑） | 逻辑被证伪，或出了相反的公告 → 走 |

### 实操建议

下单前在交易备注里写一句 **"如果___我就走"**。连这个都写不出来，说明你没想清楚，不应该下单。

## 止盈：风险收益比变了就走

养家最科学的方法——**止盈不是因为你赚了多少钱，而是因为风险和收益的对比变了。**

三种经过验证的止盈方法：

### 1. 移动止盈（最推荐）

上涨过程中不断上移你的"底线"。

**举例：**
- 盈利20%后，把底线设在盈利10%的位置
- 盈利40%后，把底线设在盈利25%的位置
- 股价回撤碰到底线就卖出，没碰到就继续持有

这样既锁定了利润，又不会卖飞主升浪。

### 2. 缩量止盈

如果上涨过程中成交量明显萎缩，说明买盘跟不上——减仓。

**背后的逻辑：** 上涨需要持续的买盘推动。量价背离是动能衰竭的信号，不该在这个时候贪。

### 3. 情绪止盈

当身边从不炒股的人都开始跟你聊股票了——该走了。

**背后的逻辑：** 市场情绪到了极端狂热的时候，后续的买盘已经枯竭了。这是养家"别人贪婪时我更贪婪"的反面——当连路人都开始贪婪了，就是该离场的时候了。

## 最常见的错误

**因为"赚了10%"就卖，结果卖飞了翻倍股。**

这不是"拿不住"的问题，而是没有系统性的止盈方法。用移动止盈来解决：

| 错误做法 | 正确做法 |
|---|---|
| 到了目标位就全卖 | 先卖一半，留一半用移动止盈 |
| 涨了怕回撤赶紧跑 | 把止损线上移，让利润奔跑 |
| 跌了死扛等反弹 | 逻辑坏了立刻走，不关心亏了多少 |

## 一句话总结

> 止损是买入时就想好的，止盈是持有中慢慢调整的。**不要在亏损时才想止损，不要在盈利时才想止盈。**

这些决策在买入那一刻就应该完成了。
