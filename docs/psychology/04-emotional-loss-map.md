# 04 · 你的情绪亏钱图谱

> 每个交易者都有自己的"亏钱模式"——特定情绪状态下一定会做的错误决策。找到它，你就找到了亏损的根源。

---

## 先连续记录一个月的情绪日志

这是找到你亏钱模式的唯一方法。不需要复杂的工具，每天收盘后花5分钟。

### 记录模板

```
日期：
持仓：
今日盈亏：
交易时的情绪（多选）：平静 / 兴奋 / 焦虑 / 恐惧 / 懊悔 / 愤怒 / 疲惫
是否按计划执行了：是 / 否
如果有未执行的计划，原因是：
一句话总结今天的心理状态：
```

### 记录原则

1. **不要评判自己的情绪**——情绪没有"好"或"坏"，它只是数据
2. **越多越准确**——10天的记录看不出什么，30天开始有规律，90天非常清晰
3. **亏损的日子尤其要详细写**——赚钱时的情绪大家都差不多，亏钱时的情绪模式才是关键

---

## 然后绘制你的情绪亏钱图谱

一个月后，拿出一张纸（或者一个表格），按以下格式整理：

### 第一步：情绪频率统计

| 情绪 | 出现次数 | 其中亏损次数 | 亏损概率 |
|---|---|---|---|
| 平静 | 15 | 4 | 27% |
| 兴奋 | 5 | 1 | 20% |
| 焦虑 | 8 | 6 | 75% |
| 恐惧 | 3 | 2 | 67% |
| 懊悔 | 6 | 5 | 83% |
| 愤怒 | 2 | 2 | 100% |
| 疲惫 | 4 | 3 | 75% |

### 第二步：找到你的"情绪雷区"

从统计中找到两个关键数据：

1. **最频繁的情绪**——你最容易在什么状态下交易
2. **亏损概率最高的情绪**——你在什么状态下交易最容易亏钱

**这就是你的情绪亏钱图谱的核心发现。**

举例：
> "我发现自己最频繁的情绪是焦虑（出现8次），而在焦虑状态下交易，亏损概率高达75%。所以我的亏钱模式是：焦虑 → 冲动交易 → 亏损 → 更焦虑 → 报复性交易 → 更大亏损。"

---

## 常见亏钱模式示例

### 模式一：焦虑型亏钱

**触发条件：** 持仓浮亏、市场波动加大、连续亏损后

**情绪链条：**
```
焦虑 → 频繁看盘 → 更焦虑 → 冲动调仓 → 做错 → 懊悔 → 想扳回来 → 再错
```

**解决方案：**
- 设置不看盘的硬规则（如：盘中只看三次）
- 提前挂好止损单，到点自动执行

### 模式二：兴奋型亏钱

**触发条件：** 大赚之后、牛市行情、看了别人的盈利截图

**情绪链条：**
```
兴奋 → 过度自信 → 加仓 → 追高 → 回调 → 亏损 → 不敢相信 → 死扛 → 大亏
```

**解决方案：**
- 大赚之后强制休息至少一天
- 设定每日/每周的仓位上限

### 模式三：懊悔型亏钱

**触发条件：** 卖飞了、错过了机会、止损后大涨

**情绪链条：**
```
懊悔 → "我不该卖" → 追回来 → 买在高点 → 再次懊悔 → 恶性循环
```

**解决方案：**
- 卖飞了就不要再看这只股票至少一周
- 接受"不可能卖在最高点"

### 模式四：疲惫型亏钱

**触发条件：** 盯盘太久、熬夜复盘、连续交易

**情绪链条：**
```
疲惫 → 注意力下降 → 忽略关键信号 → 随意交易 → 亏损 → 更疲惫
```

**解决方案：**
- 设置每天的交易截止时间
- 累了就关掉软件，强制休息

## 情绪日志的进阶用法

### 月度复盘

每个月末花半小时回顾你的情绪日志，回答三个问题：

1. **我的情绪雷区变了没有？** ——同一个问题每个月可能不同
2. **上个月做的改进效果如何？** ——认真评估，不要骗自己
3. **下个月重点改进什么？** ——每次只解决一个问题

### 季度心理报告

每季度出一次正式的心理报告，包含：

- 三个月的情绪-盈亏相关性分析
- 改进措施的执行情况
- 下一季度的心理训练计划
