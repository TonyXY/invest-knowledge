# 🧠 交易心理

> 技术可以学，但心理关必须自己过。交易心理不是"克服恐惧"那么简单，而是一套从认知到行为的系统性训练。

---

## 为什么交易心理如此重要？

<div class="insight-box">
  <div class="insight-title">📌 一个事实</div>
  <p>90%以上的交易者亏损，不是因为技术不行，而是心理关没过。</p>
  <p>技术分析可以看几本书就学会，但面对真金白银的涨跌时，能保持冷静、按计划执行的人百里挑一。</p>
</div>

交易心理的核心矛盾在于：**你的大脑是进化来在稀缺环境中生存的，不是进化来在波动的金融市场中赚钱的。**

- 面对亏损时，大脑的恐惧反应让你想"再等等"——结果等来更大的亏损
- 面对盈利时，大脑的贪婪反应让你想"再多赚点"——结果利润回吐
- 面对错失机会时，大脑的懊悔反应让你冲动追高——结果买在顶部

**这不是你意志力不行，这是人性。** 而交易的本质就是与人性对抗。
